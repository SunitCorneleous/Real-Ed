Frequently Asked Questions

It is a long established fact that a reader will be distracted by the readable content of a page when looking at its layout.

Flex is a css layout which is one dimensional. Works in either row or column. Grid is also a css layout which is multi dimensional. Works in both row and column at a time. Both of the layouts are very powerful for responsive design.

Bootstrap is a css framework which comes with pre-styled components and has countable number of utility classes. On the other hand Tailwind is a utility-first css framework which has a huge utility library to design components flexibly.

CSS box model refers to the box which wraps the html elements. Which consists margin, padding, height, width and the actual content.

Html semantic tags are html tags which gives the html document a meaningful way to express it. It helps the search engine to understand the document better. When search engine sees a article tag it understand that it is some kind of blog type of content, but if it sees only div all over the html document it can not distinguish between contents and can not give better search results.
